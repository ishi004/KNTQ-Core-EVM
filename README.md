KNTQ Arbitrage Monitor
KNTQ Arbitrage Monitor는 Hyperliquid Core와 HyperEVM 간의 KNTQ 토큰 가격 차이를 실시간으로 추적하여 차익거래 기회를 포착하는 웹 기반 도구입니다.

주요 기능

실시간 데이터 모니터링:
Hyperliquid Core: WebSocket을 통한 실시간 오더북 및 가격 수신
HyperEVM: KyberSwap Aggregator API를 통한 실시간 매수/매도 가격 조회
양방향 차익 계산: 매수/매도 방향별(HL→EVM / EVM→HL) 순수익 및 수익률 자동 계산 (수수료 반영)
수익성 분석: 오더북 깊이(Depth)를 분석하여 최대 수익 가능 물량 및 금액별($100~$5,000) 예상 수익 산출
알림 시스템: 설정한 수익률 도달 시 시각/청각 알림 제공

실행 방법
별도의 서버 설치 없이 kntq core-evm.html 파일을 브라우저에서 실행하면 즉시 사용할 수 있습니다.
